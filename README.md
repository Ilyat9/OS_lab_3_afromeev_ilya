Сборка проекта
Для сборки проекта используется утилита `make`. Проект автоматически подхватывает все исходные файлы из папки `src/`.

1. **Скомпилировать проект:**
   ```bash
   make
   
Исполняемый файл os_lab_3 появится в корневой директории.
Запустить программу:

make run

Удалить файлы сборки (очистка):

make clean
